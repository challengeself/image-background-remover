## 1. 项目概述

### 1.1 产品名称
去背景工具（Remove Background Tool）

### 1.2 产品定位
一个简单易用的 Web 图片去背景工具，用户上传图片后一键去除背景，直接下载处理后的图片。

### 1.3 目标用户
- 需要快速去除图片背景的普通用户
- 电商从业者（产品图处理）
- 设计师（素材处理）
- 自媒体从业者（配图处理）

### 1.4 核心价值
- 🚀 **快速**：一键处理，秒级响应
- 🔒 **安全**：纯内存处理，图片不落地
- 💰 **免费**：使用 remove.bg 免费额度
- 📱 **便捷**：Web 访问，无需安装

## 4. API 设计

### 4.1 去背景接口

**端点：** `POST /api/remove-bg`

**请求：**
```http
POST /api/remove-bg
Content-Type: multipart/form-data

Body:
  image: <File>  # 图片文件
```

**响应（成功）：**
```http
HTTP/1.1 200 OK
Content-Type: image/png
Content-Disposition: attachment; filename="no-bg-image.png"

<binary image data>
```

**响应（失败）：**
```http
HTTP/1.1 400 Bad Request
Content-Type: application/json

{
  "error": "错误描述"
}
```

### 4.2 错误码

| HTTP 状态码 | 错误信息 | 说明 |
|-------------|----------|------|
| 400 | 请上传图片文件 | 未提供图片 |
| 400 | 只能上传图片文件 | 文件类型错误 |
| 400 | 图片大小不能超过 10MB | 文件超限 |
| 500 | 服务器未配置 remove.bg API Key | 服务端配置问题 |
| 402 | 额度不足 | remove.bg API 额度用完 |
| 403 | API Key 无效 | remove.bg API Key 错误 |
| 500 | 服务器内部错误 | 其他异常 |

## 6. 数据流程

### 6.1 数据处理流程

```
┌─────────┐     ┌──────────┐     ┌───────────┐     ┌──────────┐
│  用户   │     │  前端    │     │  后端 API  │     │ remove.bg│
│  上传   │────▶│  读取    │────▶│  转发     │────▶│   API    │
│  图片   │     │  Base64  │     │  Buffer   │     │          │
└─────────┘     └──────────┘     └───────────┘     └──────────┘
                                         │
                                         ▼
┌─────────┐     ┌──────────┐     ┌───────────┐
│  用户   │◀────│  下载    │◀────│  处理结果  │
│  保存   │     │  Blob    │     │  Buffer   │
└─────────┘     └──────────┘     └───────────┘

⚠️ 所有数据均在内存中流转，不写入磁盘
```

### 6.2 内存管理

- 图片数据：Base64 / Blob / Buffer（内存）
- 处理后数据：Buffer（内存）
- 请求结束：自动释放（Node.js GC）
- 无持久化存储
